# yoktez-mcp

A Model Context Protocol (MCP) server for **YÖK Ulusal Tez Merkezi** (`tez.yok.gov.tr`) — Turkey's national thesis repository. Provides Turkish-aware full-text search across Master's, PhD, Medical-specialty, and Art-proficiency theses via a hybrid architecture: a baked FTS5 seed index for instant cross-thesis discovery combined with live `SearchTez` scraping to fill gaps and return up-to-date results.
